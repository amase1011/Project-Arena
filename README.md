# Project Arena v0.5

スマートフォン縦画面向けのドラフト型オートバトル・プロトタイプです。

## v0.5の主な変更

- 平原のクォータービュー風フィールド
- SD風ベクターキャラクター
- ドラフトでカードを選ぶたびにユニットを即時召喚
- 召喚されたユニットは戦闘開始までフィールドで待機
- 属性カードは「属性カード（？）」として待機
- レアカードは「レアカード（？）」として待機
- ラウンド開始時に属性・レアカードを開封
- ユニットHPバーとダメージ数値を非表示
- 30秒のカード選択制限
- 前衛・中衛・後衛の自動配置
- 爆弾兵の自爆攻撃

`index.html` を開くと遊べます。
