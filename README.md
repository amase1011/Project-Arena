# Project Arena v0.95

ブラウザで遊べる1対1ドラフト・オートバトルゲームです。  
`index.html` を開くと起動します。GitHub Pagesにもそのまま配置できます。

## v0.95 変更内容

- 同じ種類のユニットを必ず連続配置する自動整列へ変更
- 召喚や部隊2倍のたび、前衛・中衛・後衛を中央揃えで再整列
- ユニットが増えても縮小せず、同種のまとまりを維持した追加行を作成
- アサシンの攻撃力を67から50へ低下し、攻撃間隔も少し延長
- 召喚カード選択後の表示時間と再整列アニメーションを少し延長
- ラウンド終了時のライフ消失演出を延長
- ホーム画面にゲーム設定を追加
- ドラフト中の属性相性ガイドをON/OFF可能
- 設定内容をブラウザへ保存
- ラウンド結果の文字とライフを大幅に大型化
- ホーム、ドラフト、HUD、カード、ボタンなど全体の文字サイズとゲームUI感を改善

## 属性相性ガイド

設定がONの場合、ドラフト中だけ画面上部に次の有利関係を表示します。

火 → 木 → 土 → 雷 → 水 → 火

左側の属性が右側の属性に有利です。設定はホーム画面からいつでも変更できます。

## 難易度

- 初級：ランダム性が高く、属性対策は控えめ
- 中級：編成バランス、増援、前ラウンドの属性を判断
- 上級：候補を比較し、属性相性・役割不足・相手編成を総合判断

上級でも、現在のドラフト中にプレイヤーが選んだ伏せ属性は参照しません。
