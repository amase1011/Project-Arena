# Project Arena v0.4

ブラウザで動く縦画面ドラフト・オートバトルのプロトタイプです。

## 起動
`index.html` をダブルクリックしてください。

## 実装済み
- プレイヤー vs NPC
- 1回のカード選択時間30秒
- 3択×3回（敗者は次ラウンド4回）
- 引き直しは1ラウンド1回、使用後に消える
- NPCはドラフト開始時に裏で選択
- 上下HUDはハートと属性のみ
- 属性は戦闘開始時まで非公開
- 属性出現率約6%、レア合計約1%
- レアカードは開封まで中身非公開
- ライフ+1は開封時に即時反映
- 前衛・中衛・後衛の自動配置
- 個別ユニットの移動、索敵、攻撃、HP、死亡
- タンク、特攻、アサシン、アーチャー、ナイト、レンジャー、サポーター
- 爆弾兵の自爆・範囲攻撃
- 引き分けなし
